Setup (one-time)
1. Install Python 3.11+
Download from python.org → check "Add to PATH" during install.

2. Create virtual environment


cd c:\Project\trade-bot
python -m venv venv
venv\Scripts\activate
3. Install dependencies


pip install -r requirements.txt
4. Configure .env

Your .env already exists with Zerodha credentials. Open it and verify:


ZERODHA_API_KEY=your_key
ZERODHA_API_SECRET=your_secret
PAPER_TRADE=true          ← keep this true until you're confident
TRADING_CAPITAL=100000

Daily startup (every morning before market opens)
5. Generate fresh Zerodha access token (tokens expire at 6 AM IST daily)


python generate_token.py
This opens the Zerodha login in your browser → you paste the redirect URL → it saves the token to .env automatically.

Running the bot
6a. Paper trading (safe — no real orders)


python main.py --paper --symbol NIFTY
Opens dashboard at http://localhost:5000

6b. Live trading (only after testing in paper mode)


python main.py --symbol NIFTY
Will ask for confirmation before starting.

Backtesting (no broker / no token needed)
Generate sample data first (if you have no real CSV):


python backtest_run.py --generate-sample data/sample_nifty.csv --days 180
Run a backtest on the sample data:


python backtest_run.py --csv data/sample_nifty.csv
Full backtest with chart + trade log export:


python backtest_run.py --csv data/sample_nifty.csv --chart output/ --export trades.csv
Run parameter optimization (top 5 by Sharpe ratio):


python backtest_run.py --csv data/sample_nifty.csv --optimize
Backtest with real Zerodha historical data (needs valid token):


python backtest_run.py --from 2024-01-01 --to 2024-06-30
